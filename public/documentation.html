Objective

To build the server-side component of a "movies" web application.
The web application will provide users with access to information about different movies, directors, and generes.
Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Request Url HTTP Method Request body data format Response body data format
Return a list of ALL movies to the user /movies GET None A JSON object holding data about all the movies
Return data (description, genre, director, image URL, whether it's featured or not) about a signle movie by title to the user /movies/:title GET None A JSON object holding data about a single movie containing title, description, genre, image URL.
Example: {
Title: 'Iron Man',
Description: 'A man in an iron suit',
Genre: 'Action, Superhero, Thriller',
Image: 'ImageURL'
}
Return data about a genre (description) by name/title (e.g., “Thriller”). /movies/title/:genre GET None Return text message of the genre of movie selected.
Example: Iron Man is a Action, Superhero, Thriller, and Adventure movie. About a rich playboy/genius who makes a suit to become a superhero.
Return data about a director (bio, birth year, death year) by name. /movies/director/:name GET None Return information about director selected.
Example: Jonathan Kolia Favreau is an American actor, comedian and filmmaker. Born in October 19, 1966 in New York City, U.S.
Allow new users to register. /newUser POST A JSON object holding data about newUser.
Example: {
firstName: 'David',
lastName: 'Chung',
userName: 'JoonBug',
password: 'password',
email: 'email@gmail.com'
} 		A JSON object holding data about newUser that was added, including ID.
Example: {
firstName: 'David',
lastName: 'Chung',
userName: 'JoonBug',
password: 'password',
email: 'email@gmail.com',
ID: 'abc123'
}
Allow users to update their user info (username). /newUser/:username PUT None A text message indicating the user's info was updated.
Allow users to add a movie to their list of favorites. /newUser/username/:favorites POST A JSON object holding data about a single movie containing title, genre, director, release year.
Example: {
Title: 'Captain Marvel',
Genre: 'Action, Sci-fi',
Director: 'Anna Boden, Ryan Fleck',
Year: '2019'
} 		A text message indicating user sucessfully added movie to their favorites.
Allow users to remove a movie from their list of favorites. /newUser/username/:favorites DELETE None A text message indicating user sucessfully removed movie from their list of favorites.
Allow existing users to deregister. /newUser DELETE None A text message indicating user sucessfully removed/deleted their account.
